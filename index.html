Cognition Project

Overview

The Cognition Project is a web-based application designed to track and analyze motor performance through touch interactions. It features user authentication, role-based access (participants and caregivers), and a RESTful API for handling measurements and results. The system is built with Node.js, Express, and PostgreSQL.

Features

  • Motor Performance Metrics: Tracks drag speed, distance, acceleration, and tap duration.
  • User Roles: Participants and caregivers with specific permissions.
  • API Integration: RESTful API for managing users, measurements, and results.
  • Database Storage: Uses PostgreSQL for data persistence.

How to Run

  1. Install Node.js and PostgreSQL.
  2. Clone the repository.
  3. Install dependencies: npm install
  4. Set up the PostgreSQL database.
  5. Start the server: npm run
  6. Open http://localhost:8080 in a browser.

API Endpoints

Authentication

  • POST /auth/login
  • POST /auth/logout

Measures

  • POST /measures/save
  • GET /measures/:id
  • PUT /measures/:id
  • DELETE /measures/:id

Results

  • GET /results/calculateResult
  • POST /results/submitdata
